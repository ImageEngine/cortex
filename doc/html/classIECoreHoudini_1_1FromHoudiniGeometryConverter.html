Detailed Description

The FromHoudiniGeometryConverter class forms a base class for all classes able to perform some kind of conversion from a Houdini GU_Detail to an IECore::Primitive.

IECore::TypeId  resultType = IECore::InvalidTypeId 
) [static]

Creates a converter which will convert the given Houdini GU_Detail to an IECore::Primitive. If resultType is specified then only converters which create objects of that type will be returned - the default value allows any suitable converter to be created. If no matching converters exist then returns 0.

virtual IECore::ObjectPtr IECoreHoudini::FromHoudiniGeometryConverter::doConversion ( IECore::ConstCompoundObjectPtr  operands  )  const [protected, virtual]

Implemented to aquire the read lock on the GU_Detail held by the GU_DetailHandle, call doPrimitiveConversion(), and finally unlock the GU_Detail. Derived classes need not reimplement this function, but should instead implement doPrimitiveConversion().
