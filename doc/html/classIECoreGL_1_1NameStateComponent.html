Detailed Description

The NameStateComponent class is used to specify the names of objects being rendered. It maps from a public name stored as a string to a private integer name which is specified to OpenGL through glLoadName(). It then provides a means of examining an OpenGL selection buffer and returning the integer names in their string form.
