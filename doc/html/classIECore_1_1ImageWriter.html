StringParameter* IECore::ImageWriter::colorspaceParameter (  ) 

The parameter specifying the colorspace that the given image will be when stored in the file. If autoDetect is chosen than it will use the colorspace returned by destinationColorSpace(). The input image is assumed to be in linear colorspace.

virtual std::string IECore::ImageWriter::destinationColorSpace (  )  const [pure virtual]

Returns the name of default colorspace in which the Writer expects to receive images. The base class is responsible for making sure it will happen.
