Detailed Description

Defines functions to allow the RenderMan Dspy interface to be implemented in terms of IECore::DisplayDrivers. These functions are automatically registered with RenderMan using DspyRegisterDriverTable, but are public to allow their use in implementing display driver dsos as well.
