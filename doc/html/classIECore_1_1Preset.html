Detailed Description

The Preset class serves as a base class for the implementation of 'presets'.

In a nutshell, they are callable classes that manipulate the parameters of a Parameterised object.

See also:
BasicPreset for an implementation that provides parameter value loading an saving, with support for Class and ClassVector parameters.

Presets themselves are Parametersied objects, to allow them to have their own parameters to control how they might be applied.
