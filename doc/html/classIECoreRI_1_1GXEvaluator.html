Detailed Description

This class simplifies the use of the Gx API by wrapping it to accept IECore datatypes for input and output.

Evaluates points at the specified s,t positions. As an individual evaluation can fail if no geometry exists at that location, an additional "gxStatus" BoolVectorData is returned, with elements being true only if the corresponding results are valid.

Threading:
It is safe to call this from multiple concurrent threads.
