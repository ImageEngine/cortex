Detailed Description

The Renderable class provides an abstract base class for all classes capable of producing a visible result in an OpenGL context.

Member Function Documentation

virtual void IECoreGL::Renderable::render ( const State *  state  )  const [pure virtual]

Renders the object in the current OpenGL context, using the state specified. The state should already have been bound into the current context, and state->isComplete() must be true. Upon leaving render() the OpenGL state is unchanged.
