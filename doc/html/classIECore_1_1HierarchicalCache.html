Detailed Description

A simple means of storing VisibleRenderable objects in a hierarchical way. Uses an IndexedIOInterface object to access the file. The file is organized as following:

  • /headersH/<headerName> (Object)
  • /children/<objName>/attributes/<attrName>
  • /children/<objName>/boundingBox (Box3f)
  • /children/<objName>/transformMatrix (M44f) or /children/<objName>/shape/ ( VisibleRenderable ).
  • /children/<objName>/children/... Will throw an exception derived from IEException if any errors are encountered.
