Detailed Description

A base class for warp operations on ImagePrimitive objects. This Op modifies an image by remapping pixels values to other locations. The display window does not change in this process, but the data window may change. The mapping is determined by the derived classes. The base class is responsible for resizing the data window and applying filter on the colors based on the floating point positions returned by warp method.
