Detailed Description

The Converter class forms a base class for all classes able to perform some kind of conversion between 3rd party types and IECore datatypes. It derives from Parameterised and uses the parameter values to define how conversion is performed. For fast conversions of very simple objects where there are no parameters necessary (for example vector types) see Convert.h.
