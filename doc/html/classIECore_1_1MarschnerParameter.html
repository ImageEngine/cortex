Detailed Description

The MarschnerParameter provides a convenience class to handle the various parameters needed to evaluate the Marschner hair shading model implemented in the Marschner.h. This parameter is specialised to T = Color3f. It provides the option to present the Absorption parameter of the model as a color. Defaults are derived from the relevant papers.
