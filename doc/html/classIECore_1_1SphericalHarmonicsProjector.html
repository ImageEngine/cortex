Detailed Description

template<typename V>
class IECore::SphericalHarmonicsProjector< V >

Class that projects functions on spherical harmonics objects. The object defines a distribution of points over the sphere surface and uses that distribution to project a given function onto a spherical harmonics object. Based mainly on "Spherical Harmonic Lighting: The Gritty Details" by Robin Green.
