Detailed Description

The RATDeepImageWriter class writes Houdini deep texture files. Note that C will only be added to the RAT file if RGBA channels are specified in order. As Of is required by the format, A will always be converted to an Of triple. If A is not provided, a value of 1.0 will be used for all Of sub-channels.
