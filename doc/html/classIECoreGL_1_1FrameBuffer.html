Detailed Description

The FrameBuffer object provides a nice reference counted wrapper around the OpenGL framebuffer object extension. It uses the Texture classes to set the components of the framebuffer.

Constructor & Destructor Documentation

IECoreGL::FrameBuffer::FrameBuffer (  ) 

Makes a new framebuffer. At this point the buffer is empty - you must use the set*() functions below to provide locations to draw to before using it. Throws an Exception if framebuffers are not supported by the OpenGL implementation.
