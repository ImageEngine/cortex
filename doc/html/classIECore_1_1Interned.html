Detailed Description

template<typename T, typename Hash = Hash<T>>
class IECore::Interned< T, Hash >

The Interned class provides a means of efficiently storing multiple different objects with the same value. It does this by keeping a static table with the actual values in it, with the object instances just referencing the values in the table.
