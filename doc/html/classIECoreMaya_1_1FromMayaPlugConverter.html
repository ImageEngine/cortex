Detailed Description

The FromMayaPlugConverter class allows conversion from MPlug values to an IECore Object.

Member Function Documentation

static FromMayaConverterPtr IECoreMaya::FromMayaPlugConverter::create ( const MPlug &  plug,
IECore::TypeId  resultType = IECore::InvalidTypeId 
) [static]

Creates a converter to convert the plug value to the an IECore type. If resultType is specified then only converters which would create that type will be considered. May return 0 if no converter can be found. Note that this returns a generic FromMayaConverterPtr rather than a FromMayaPlugConverterPtr so that a FromMayaObjectConverterPtr may be returned for the case of a plug holding an MObject.
