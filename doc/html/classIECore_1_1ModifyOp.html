Detailed Description

The ModifyOp class defines a base class for Ops whose result is some modification of an input object. It defines a parameter for the input object, a parameter to define whether or not the object is modified in place or is copied and a parameter to disable the operation completely. It's a little bit naughty to modify it in place but it'll probably be quite handy at times.
