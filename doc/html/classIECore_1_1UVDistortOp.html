virtual Imath::V2f IECore::UVDistortOp::warp ( const Imath::V2f &  p  )  const [protected, virtual]

Called once per element (pixel for ImagePrimitives). Must be implemented by subclasses to determine where the color will come from. The returned coordinate is on pixel space of the input image and the given V2f coordinates are on the output image pixel space.

Implements IECore::WarpOp.
