Detailed Description

The MatrixMultiplyOp applies a matrix transformation to an array of vectors. It can transform the vectors as if they were points, vectors or normals.

Member Enumeration Documentation

enum IECore::MatrixMultiplyOp::Mode

An enumeration for the different ways of interpreting the vectors to be transformed. These are the acceptable values for modeParameter()->setTypedValue().
