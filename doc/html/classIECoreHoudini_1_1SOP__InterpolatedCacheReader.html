Detailed Description

SOP for applying an IECore::InterpolatedCache to the incoming Houdini geometry. The GB_PointGroups found on the incoming GU_Detail will be combined with the Object Prefix/Suffix parameters to form the IECore::InterpolatedCache::ObjectHandles. If IECore::InterpolatedCache::AttributeHandles exist for an ObjectHandle, they will be added to the GU_Detail as a GB_Attribute and the values will be transfered for the GB_Elements contained within the GB_PointGroup. The GB_Attribute name will be the difference between the AttributeHandle and the Attribute Prefix/Suffix parameters.
