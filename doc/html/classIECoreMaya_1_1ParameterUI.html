Detailed Description

Base class for objects which are able to create an Attribute Editor widget for a single IECore.Parameter held on an IECoreMaya.ParameterisedHolder node.

Todo:
Separate control drawing from labelling and layout, so these classes just create the right hand side of what they're doing at the moment. Then we can use them in different layouts like spreadsheets and wotnot.

The parameterisedHolderNode is an MObject specifying the node holding the specified IECore.Parameter.

Derived class __init__ implementations must create a layout to hold all their contents and pass this in the topLevelUI parameter (as for all UIElement derived classes).

def IECoreMaya::ParameterUI::replace (   self,

Derived classes should override this method.

The override should first call the base class method and then reconnect all created widgets to the new node/parameter. The node and parameter arguments are as for the __init__ function.

Reimplemented in IECoreMaya::StringParameterUI, IECoreMaya::NoteParameterUI, IECoreMaya::ClassVectorParameterUI, IECoreMaya::PresetsOnlyParameterUI, IECoreMaya::StringVectorParameterUI, and IECoreMaya::TransformationMatrixParameterUI.
