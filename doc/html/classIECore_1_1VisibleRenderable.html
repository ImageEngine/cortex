Detailed Description

The VisibleRenderable class is an abstract base class for all Renderables which describe a visible object (primitive shape, procedural etc) to a Renderer. VisibleRenderables guarantee to leave the Renderer state (attributes, transforms, shaders) unchanged across calls to VisibleRenderable::render().
