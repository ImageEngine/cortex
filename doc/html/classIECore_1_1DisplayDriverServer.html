Detailed Description

Server class that receives images from ClientDisplayDriver connections and forwards the data to local display drivers. The type of the local display drivers is defined by the 'remoteDisplayType' parameter.

The server object creates a thread to control the socket connection. The thread dies when the object is destroyed.
