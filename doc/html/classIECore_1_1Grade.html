Detailed Description

The Grade implements the same operation as Nuke's grade node over the colors of a Primitive object. The computation performed is: A = multiply * (gain - lift) / (whitePoint - blackPoint) B = offset + lift - A * blackPoint output = pow( A * input + B, 1/gamma )
